Dit is de source van de website van Koor Kaskedieze.

./httpdocs bevat de door iWeb gegenereerde oude shit.
Deze is niet langer in gebruik wegens ontbrekende ondersteuning door Apple sedert 2012.

./kasketest is de nieuw opgezette site op basis van HTML5 en CSS3.
Voor de het foto-gedeelte gebruiken we fancybox.

ENGLISH
=======
This contains the source of the website of the Dutch Choir Kaskedieze.

.httpdocs retains for the moment the shit generated by the iWeb application.
I used it until recently to maintain the website.
Until I discovered Apple no longer supported it, since 2012 already.

./kasketest is the new website based on HLMT5 and CSS3,
using fancybox javascript plugin for the foto part.
