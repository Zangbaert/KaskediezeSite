ENGLISH

This contains the source of the website of the Dutch Choir Kaskedieze.

I used to maintain the website with the wonderful iWeb product.
Until I discovered Apple no longer supported it, since 2012 already.

The new website is based on HLMT5 and CSS3,
using fancybox javascript plugin for the foto part.
